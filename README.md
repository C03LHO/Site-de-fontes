# 🎨 Site de Fontes

Um site simples e elegante para explorar, testar e favoritar as melhores fontes do Google Fonts.

## ✨ Funcionalidades

- **70+ fontes famosas** disponíveis para teste
- **Caixa de texto** para digitar e visualizar em tempo real
- **Controle de tamanho** da fonte (12px a 120px)
- **Seletor de cor** para a letra
- **Marca texto** com cor personalizável (pode ser ativado/desativado)
- **Sistema de favoritos** com estrela ⭐
- **Histórico** das últimas 20 fontes usadas
- **Filtro** para ver todas as fontes ou apenas favoritas
- **Persistência local** - todas as preferências são salvas no navegador (localStorage)

## 🚀 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Digite seu texto na caixa de texto
3. Clique em qualquer fonte da lista para aplicá-la
4. Use os controles para ajustar tamanho, cor e marca texto
5. Clique na estrela (☆) para favoritar uma fonte
6. Use o filtro "Favoritas" para ver apenas suas fontes preferidas
7. Clique em qualquer item do histórico para reutilizar uma fonte

## 📦 Deploy no GitHub Pages

1. Crie um repositório no GitHub
2. Faça upload dos arquivos:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Vá em **Settings** > **Pages**
4. Selecione a branch `main` (ou `master`)
5. Clique em **Save**
6. Seu site estará disponível em: `https://seu-usuario.github.io/nome-do-repositorio/`

## 🎯 Tecnologias Utilizadas

- HTML5
- CSS3 (com Grid e Flexbox)
- JavaScript (Vanilla JS)
- Google Fonts API
- LocalStorage para persistência

## 📝 Notas

- Todas as preferências são salvas localmente no navegador
- Não há necessidade de login ou backend
- Funciona completamente offline após o primeiro carregamento
- Totalmente responsivo e compatível com dispositivos móveis
